 

### 进入D盘

如果要进入d盘

```bash
d:
cd D:\Program Files\Microvirt\MEmu\
```

输出结果如下：

```
C:\Users\zhaoq>d:
D:\>
```

而如果通过  ` cd d:\ ` 却无法进入D盘。

```bash
C:\Users\zhaoq>cd d:\

C:\Users\zhaoq>
```



### 进入C盘

同样通过上面的方法，却无法进入到根目录

```bash
C:\Users\zhaoq>c:

C:\Users\zhaoq>c:
```

需要使用 `cd c:\`

```bash
C:\Users\zhaoq>cd c:\
c:\>
```
